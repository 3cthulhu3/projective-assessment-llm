# Projective Assessment Battery — LLM Guided

This repository contains a prompt for administering:

- Rorschach Inkblot Test (Exner-based structure)
- Thematic Apperception Test (TAT)

The prompt guides an LLM to:

- Administer tests
- Record responses
- Score responses
- Provide integrated psychodynamic analysis

## Warning

This is NOT a clinical tool.  
Not for diagnosis.  
For research and exploration only.

## Usage

Paste the prompt into an LLM and follow instructions.
